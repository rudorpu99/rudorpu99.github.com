복합 대입 연산자

연산자 설명
+= 기존 변수의 값에 값을 더합니다.
-= 기존 변수의 값에 값을 뺍니다.
*= 기존 변수의 값에 값을 곱합니다.
/= 기존 변수의 값에 값을 나눕니다.
%= 기존 변수의 값에 나머지를 구합니다.

증감 연산자

연산자 설명
변수++ 기존 변수 값에 1을 더합니다.(후위)
++변수 기존 변수의 값에 1을 더합니다.(전위)
변수-- 기존 변수 값에 1을 뺍니다.(후위)
--변수 기존 변수의 값에 1을 뺍니다.(전위)
